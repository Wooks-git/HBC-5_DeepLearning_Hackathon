# HBC-5_DeepLearning_Hackathon
2020 Hallym Hackathon

# 대회 설명

교내 건물 (공학관, 일송아트홀, 산학협력관, 대학본부 별관, CLC 총 5개의 class)의 데이터를 직접 수집하여, <br>

수집한 사진을 분류하는 딥러닝 모델을 개발.

# evaluation
